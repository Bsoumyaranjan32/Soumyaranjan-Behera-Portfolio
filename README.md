# Portfolio — Soumyaranjan Behera

This is a personal portfolio built with HTML, Tailwind CSS and vanilla JavaScript.

## Setup

Install dependencies (already added Tailwind CLI):

```bash
npm install
```

Build Tailwind CSS once:

```bash
npm run build:css
```

Or watch for changes during development:

```bash
npm run watch:css
```

Then open `index.html` in your browser.

## Features

- Responsive layout using Tailwind CSS
- Mobile navigation toggle
- Theme toggle (dark / light preference persisted)
- Typed.js effect in hero (CDN included)
- Contact form wired to FormSubmit

## Notes

- If you want a downloadable resume, add `resume.pdf` to the project root. The "Download CV" button will open it, otherwise it falls back to your GitHub page.
- To deploy, you can push to GitHub Pages or any static hosting provider.
